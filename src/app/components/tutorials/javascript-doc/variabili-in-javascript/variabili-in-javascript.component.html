Prima dell'introduzione nel 2015 di ES6 esisteva un solo modo per dichiarare le variabile, ossia con la parola chiave var.

Tra le differenze principali che ci sono tra queste 2 keywords c'è lo scope, che può essere locale, globale, funzionale o di tipo blocco.

Analizziamo una alla volta tutte e 3 le parole chiave.

VAR

Come già detto sopra, prima del 2015 esisteva solo la parola chiave var per dichiarare le variabile.

Per quanto riguarda var, il suo scope, se dichiarata dentro ad una funzione, si parla di functional scope, ciò significa che la variabile può essere usata solo all'interno della funzione in cui è stata dichiarata e quindi questo uquivale anche a dire che la variabile ha un local scope (la funzione).
